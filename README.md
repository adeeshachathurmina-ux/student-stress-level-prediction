# Student Stress Level Prediction

This project was completed for the Introduction to Data Science coursework.

## Project Topic

Predicting student stress level based on social media usage and lifestyle factors.

## Dataset

The dataset used in this project is the Student Social Media & Mental Health Impact Dataset from Kaggle.

Dataset link:
https://www.kaggle.com/datasets/shivasingh4945/student-social-media-and-mental-health-impact

## Target Variable

Stress_Level

The target variable has four categories:

- Low
- Medium
- High
- Very High

## Models Used

The following classification models were used:

- K-Nearest Neighbours
- Logistic Regression
- Support Vector Machine

## Model Accuracy

| Model | Accuracy |
|---|---|
| K-Nearest Neighbours | 0.759 |
| Logistic Regression | 0.751 |
| Support Vector Machine | 0.745 |

## Best Model

K-Nearest Neighbours was selected as the best model because it achieved the highest accuracy.
